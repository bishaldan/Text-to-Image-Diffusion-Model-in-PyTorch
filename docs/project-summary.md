## Slightly Longer Summary

This project is a small educational text-to-image diffusion model designed to make AI image generation understandable. It includes a synthetic captioned dataset generator, a DDPM-style conditioned U-Net, prompt conditioning with a frozen BERT encoder, reproducible training and sampling workflows, Docker support, tests, and visual documentation aimed at both recruiters and learners.
